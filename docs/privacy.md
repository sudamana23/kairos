# Privacy Policy — Tenets

**Last updated: April 2026**

Tenets does not collect, transmit, or store any personal data.

All information you enter — values, domains, objectives, key results, 
pulse entries, and reviews — is stored exclusively on your Mac using 
Apple's local SwiftData framework. Nothing is sent to any server, 
third party, or cloud service.

There is no account, no login, and no analytics. The app does not 
use advertising frameworks or tracking of any kind.

If you use the Export Backup feature, the resulting file is saved 
locally to a location you choose. Tenets does not transmit this file 
anywhere.

**On-device AI.** If your device supports Apple Intelligence, Tenets 
may use Apple's on-device LanguageModelSession API to generate 
suggestions. This processing happens entirely on your device. No 
text you enter is sent to Apple or any other service as a result of 
using these features.
